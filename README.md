# learn-open-gl-v2
Learn OpenGL exercises, compiling with cmake cross-platform (Windows and Mac) 

## Hey, kids! cmake is fun! 
1. Install glfw. Mac:

```
    brew install glfw
```
Windows TBD

2. `mdkir build`
3. `cd build`
4. `cmake ..`
5. `make`
6. `./learn-open-gl-v2`

That will run the program.
